# Login Page

A simple and responsive Login Page built using HTML and CSS. This project demonstrates the fundamentals of creating a clean user authentication interface with a modern and user-friendly design.

## Features

* Responsive layout
* Clean and modern UI
* Email and Password input fields
* Login button
* "Forgot Password?" link
* Sign Up link
* Mobile-friendly design

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
login-page/
│── index.html
│── style.css
└── images/
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
